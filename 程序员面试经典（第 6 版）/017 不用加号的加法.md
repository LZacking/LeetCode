### 面试题 17.01. 不用加号的加法     
设计一个函数把两个数字相加。不得使用 + 或者其他算术运算符。


#### 示例:             
输入: a = 1, b = 1          
输出: 2

****


### 代码
```java
class Solution {
    public int add(int a, int b) {
        while (b != 0) {
        int temp = a ^ b;
        b = (a & b) << 1;
        a = temp;
    }
        return a;
    }
}
```
